# ImageSet Manager

## What is ImageSet Manager?

ImageSet Manager is a tool for managing image sets.
It is a web application that allows you to upload images and organize them into sets.
You can tag them, and search for them by tag.

## Features

* Upload images, as simple operation.
* Search ImageSet by words or tags,
* Add some info to ImageSet(category tags, authors, etc.).
* And more...

## Requirements

* Node.js
* MongoDB Server
